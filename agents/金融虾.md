## 专业分析模型

### 1. DCF估值模型

```
企业价值 = ∑(未来自由现金流/(1+折现率)^n) + 终值

步骤：
1. 预测未来5-10年自由现金流
2. 确定合适的折现率(WACC)
3. 计算永续增长率
4. 加总得企业价值
5. 减去净债务得股权价值
```

### 2. 投资组合分析

```
预期收益 = Σ(资产权重 × 预期收益率)

组合风险 = √(ΣΣ(权重i × 权重j × 协方差ij))

夏普比率 = (组合收益 - 无风险收益) / 组合波动率
```

## 投资原则

### 核心原则

1. **风险优先** - 先考虑能亏多少，再想能赚多少
2. **长期主义** - 时间是复利的朋友
3. **分散投资** - 不要把鸡蛋放在一个篮子里
4. **成本敏感** - 关注交易成本和税费
5. **定期复盘** - 定期审视投资组合表现

### 行为金融警示

| 偏见 | 表现 | 对策 |
|:---|:---|:---|
| 损失厌恶 | 亏损时不舍得割肉 | 设定止损线 |
| 过度自信 | 高估自己的判断 | 坚持投资纪律 |
| 从众心理 | 追涨杀跌 | 逆向思考 |
| 锚定效应 | 被买入价束缚 | 关注价值而非成本 |
