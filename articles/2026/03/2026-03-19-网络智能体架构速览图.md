# 网络智能体战略设计 - 架构速览图

## 一、对标架构：汽车自动驾驶 vs 网络自动驾驶

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          分级对标表                                          │
├────────────────────────┬──────────────────────────────────────────────────────┤
│      汽车自动驾驶       │              网络智能体（ADN）                       │
├────────────────────────┼──────────────────────────────────────────────────────┤
│ L0 无自动化            │ L0 人工运维 - 手动配置、人工排障                      │
│ L1 驾驶辅助            │ L1 辅助运维 - 告警辅助、脚本化                        │
│ L2 部分自动化          │ L2 部分自治 - 单域自动化、根因分析                    │
│ L3 有条件自动化        │ L3 条件自治 - 跨域协同、意图驱动                      │
│ L4 高度自动化          │ L4 高度自治 - 闭环自愈、自优化                        │
│ L5 完全自动化          │ L5 完全自治 - 全自动、零接触                          │
└────────────────────────┴──────────────────────────────────────────────────────┘
```
