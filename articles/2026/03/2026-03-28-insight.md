## 二、LLM 推理效率：扩散解码与 Token 压缩同时突破

### 2.1 扩散 LLM 解码：S2D2 实现 4.7x 加速

arXiv:2603.25702 提出的 S2D2 利用 block-diffusion 模型退化为自回归（block size=1）的特性，让同一预训练模型同时充当 drafter 和 verifier，在 SDAR 上实现 **4.7x 加速**，LLaDA2.1-Mini 上比静态基线快 4.4x 且精度略高。零训练成本，混合并行扩散 + 局部序列级 AR critic 解码轨迹。

这是扩散语言模型推理效率的重大突破，意味着非自回归生成路径在保持精度的同时获得了实质性加速。

### 2.2 Token 压缩：18x 压缩仍保留下游性能

arXiv:2603.25340 提出的自表达自编码框架，将预训练 LLM 微调为将长文本压缩为离散可变长潜在码（Z-tokens）。内容自适应：语义密集段获得更多 Z-tokens，可预测区域被激进压缩。Wikipedia 等数据集上实现 **最高 18 倍 token 压缩**，支持提示压缩和 Z-token 空间内自回归生成。

**工程含义**: 长上下文推理的 token 高效路径已经打通，18x 压缩下仍保留下游性能，对 RAG 和 Agent 记忆系统有直接价值。

### 2.3 自改进 LLM 综述

arXiv:2603.25681 提出统一闭环框架，模型在各阶段均发挥核心作用：数据生成器、选择信号、参数更新器、输出精炼器。面向完全自改进 LLM 的系统性技术路线图已经成型。

## 三、Agent 评估：从刷分到结构化可信度验证

### 3.1 深度研究 Agent 的严峻现实

arXiv:2603.25342 提出范畴论形式化框架，将深度研究工作流建模为保结构映射的组合。11 个主流模型评估结果：SOTA 仅 **19.9% 平均准确率**。多跳结构综合仍是 Agent 的严峻挑战——几乎全部模型在拓扑重排序和本体验证上崩溃。这是当前 Agent 能力天花板的精确测量。

### 3.2 PICon：Persona Agent 一致性危机

arXiv:2603.25620 提出 PICon 框架，通过逻辑链多轮追问探测 Agent 三维一致性：内部、外部、重测。63 名真实人类对比：即使先前报告"高度一致"的系统，在所有三维上均未达到人类基线，揭示矛盾和回避行为。

**结论**: LLM persona agent 的"人格一致性"是幻觉，系统化追问暴露了根本性局限。

### 3.3 跨模型不确定性检测

arXiv:2603.25450 提出 Cross-Model Perplexity (CMP) 和 Cross-Model Entropy (CME)，在"自信错误"这一最危险失败模式下，用第二个验证模型对生成模型答案做单次前向传播，测量惊讶程度。MMLU 上 AUROC 从 0.59 提升至 **0.75**。

免训练、免标签，直接插入生产路由/监控管线。

### 3.4 WildASR：语音 Agent 场景的 ASR 鲁棒性风险

arXiv:2603.25727 揭示现有 ASR 系统在真实语音 Agent 场景中仍有严重且不均匀的性能退化。关键风险：模型在部分/降级输入下会幻觉性生成流畅但完全错误的内容，带来下游 Agent 安全风险。
