## 四、多平台监控策略

### 4.1 Hacker News

| 维度 | 策略 |
|:---|:---|
| 监控内容 | Top 30 + 包含关键词的帖子 |
| 关键词 | ai, llm, gpt, startup, funding, chip, gpu |
| 抓取方式 | hnrss.org/newest（RSS） |
| 归档 | `daily-news/hacker-news/YYYY-MM-DD.md` |

### 4.2 GitHub

| 维度 | 策略 |
|:---|:---|
| 监控内容 | Trending + 新项目（stars > 100） |
| 关键词 | ai, llm, agent, rag, mlops |
| 抓取方式 | exa_web_search + GitHub API |
| 归档 | `daily-news/github-trending/`（已有） |

### 4.3 arXiv

| 维度 | 策略 |
|:---|:---|
| 监控内容 | cs.CL, cs.AI, cs.LG 分类新论文 |
| 筛选 | 引用 > 10 或 Google Scholar 收录 |
| 抓取方式 | arXiv RSS（已有） |
| 归档 | `daily-news/arxiv/`（已有） |
