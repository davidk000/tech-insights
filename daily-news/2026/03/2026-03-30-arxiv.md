### 1. MemBoost: Cost-Aware LLM 推理的 Memory-Boosted 框架
**arXiv:2603.26557** | Authors: Zixuan Liu et al. | cs.CL

**摘要**：大型语言模型（LLM）在实际服务中推理成本高昂，尤其是跨用户和会话的重复或近似查询。MemBoost 是一个 memory-boosted LLM serving 框架，让轻量级模型复用历史答案，仅将困难/不确定查询升级到强模型。与标准 RAG 不同，MemBoost 支持交互式场景下的答案复用、持续 memory 增长和成本感知路由。实验表明显著减少大模型调用次数和总体推理成本，同时保持与强模型基线相当的答案质量。

**PDF**：[https://arxiv.org/pdf/2603.26557](https://arxiv.org/pdf/2603.26557)

### 2. When Perplexity Lies: Hybrid-KDA 架构 + GenDistill 蒸馏
**arXiv:2603.26556** | Authors: Juan Kostelec et al. | cs.CL / cs.AI

**摘要**：将预训练 Transformer 通过蒸馏转化为更高效的混合模型是降低推理成本的有效方法。但很多蒸馏工作用 log-likelihood 而非自回归生成来评估下游多选题，导致结论误导——例如 7B 参数蒸馏模型在 perplexity 上与教师模型差距仅 0.2pp，自回归生成却落后 20.8pp。本文提出 Hybrid-KDA 架构和 GenDistill 多阶段蒸馏流水线，全程用生成式评估引导设计决策。最佳 Hybrid-KDA 模型在知识基准上保留教师 86-90% 准确率，KV cache 减少 75%，128K token 上下文下首 token 时间提升 2-4×。

**PDF**：[https://arxiv.org/pdf/2603.26556](https://arxiv.org/pdf/2603.26556)

### 3. Weight Tying 对 Token Embeddings 的输出偏移
**arXiv:2603.26663** | Authors: Akshat Gupta et al. | cs.CL

**摘要**：Weight tying（输入输出嵌入矩阵参数共享）是语言模型常见实践，但其对嵌入空间的影响尚不清晰。本文证明 tied embedding 矩阵与 output (unembedding) 矩阵的对齐程度远高于与 input embeddings 的对齐，说明共享矩阵主要服务于输出预测而非输入表征。这种 "unembedding bias" 源于训练早期 output gradients 主导。使用 tuned lens 分析表明这负面影响早期层计算。 Scaling input gradients 可减轻该偏移，为理解 weight tying 在规模化场景下为何损害性能提供了因果证据。

**PDF**：[https://arxiv.org/pdf/2603.26663](https://arxiv.org/pdf/2603.26663)

### 5. ALBA: 欧洲葡萄牙语 LLM 评测基准
**arXiv:2603.26516** | Authors: Rafael Ferreira et al. | cs.CL / cs.AI / cs.LG

**摘要**：随着 LLM 扩展到多语言领域，评估代表性不足语言的模型性能变得越来越重要。欧洲葡萄牙语（pt-PT）受影响尤为严重——现有训练数据和基准主要来自巴西葡语。本文提出 ALBA，一个基于语言学设计的评测基准，从零构建评估 pt-PT 语言能力的 8 个维度：语言变体、文化绑定语义、话语分析、文字游戏、句法、形态学、词汇学、音系学。ALBA 由语言专家手动构建，配合 LLM-as-a-judge 框架实现可扩展评测。在多样化模型上的实验揭示了各维度性能差异，强调了支持 pt-PT 工具开发的综合变体敏感基准的必要性。（PROPOR 2026）

### 7. Muon 优化器的存储容量分析
**arXiv:2603.26554** | Authors: Juno Kim et al. | cs.LG

**摘要**：Spectral 优化器如 Muon 在大规模 LLM 训练中表现出色，但其优势来源和范围尚不清楚。本文通过线性联想记忆问题研究该问题（beyond 正交嵌入，采用 Gaussian 输入/输出）。主要结果精确刻画了 logistic 损失下 Muon 和 SGD 在 power law 频率分布下的单步恢复率：Muon 存储容量显著超过 SGD，且在更大临界 batch size 下饱和。多步动力学分析表明 Muon 初始恢复率远快于 SGD，两者最终以相当速度收敛到信息论极限。合成任务实验验证了预测的 scaling laws。

**PDF**：[https://arxiv.org/pdf/2603.26554](https://arxiv.org/pdf/2603.26554)

### 8. EU 药物警戒信号检测的时间索引参考数据集

### 9. AIRA²: 克服 AI 研究 Agent 的结构性瓶颈
**arXiv:2603.26499** | Authors: Karen Hambardzumyan, Yoram Bachrach, Jakob Foerster 等 (27人) | cs.AI

**摘要**：现有研究识别出 AI 研究 Agent 三个结构性性能瓶颈：(1) 同步单 GPU 执行限制样本吞吐量；(2) 基于验证的选择导致性能随搜索时间延长而下降的泛化差距；(3) 固定单轮 LLM 算子的能力上限。AIRA² 通过三个架构选择解决这些问题：异步多 GPU worker pool（线性提升吞吐量）、Hidden Consistent Evaluation 协议（提供可靠评估信号）、ReAct agents（动态 scoping 动作并交互式调试）。在 MLE-bench-30 上，AIRA² 在 24 小时达到平均 Percentile Rank 71.8%（超越此前最佳 69.9%），72 小时稳步提升至 76.0%。消融研究表明每项组件均必要，"过拟合" 实为评估噪声驱动。

**PDF**：[https://arxiv.org/pdf/2603.26499](https://arxiv.org/pdf/2603.26499)

### 10. CADSmith: 多 Agent CAD 代码生成 + 程序化几何验证
**arXiv:2603.26512** | Authors: Jesse Barkley et al. | cs.AI

**摘要**：现有 text-to-CAD 生成方法要么单次无验证，要么依赖有损视觉反馈无法解决维度误差。CADSmith 是一个多 Agent 流水线，从自然语言生成 CadQuery 代码后经过两重修正循环：内环解决执行错误，外环基于 OpenCASCADE 内核的精确测量（边界框尺寸、体积、实体有效性）和 VLM Judge 的整体视觉评估进行程序化几何验证。系统使用 RAG 检索 API 文档而非微调。评测表明：执行率从 95% 提升到 100%，中位 F1 从 0.9707 提升到 0.9846，IoU 从 0.8085 提升到 0.9629，Chamfer Distance 从 28.37 降低到 0.74。

### 11. 不确定性感知的多 Agent 澄清策略（LLM Coding Agents）

### 12. JAL-Turn: 语音对话系统中的联合声学-语言建模
**arXiv:2603.26515** | Authors: Yu Pan et al. | cs.CL / cs.AI

**摘要**：高效鲁棒的 turn-taking 检测在工业级 Voice AI Agent 部署中仍是重大挑战。很多现有系统依赖纯声学或语义线索，导致准确率和稳定性欠佳；而最近赋予 LLM 全双工能力的尝试需要昂贵的全双工数据，训练和部署开销大。本文提出 JAL-Turn，一个轻量级纯语音 turn-taking 框架，采用联合声学-语言建模范式，跨注意力模块自适应整合预训练声学表示与语言特征，以低延迟预测 hold vs shift 状态。通过共享冻结 ASR 编码器，JAL-Turn 与语音识别完全并行运行，不引入额外端到端延迟或计算开销。公开多语言基准和日语客服数据集上均超越 SOTA 基线，同时保持优越实时性能。

### 13. Switch Attention: 动态细粒度混合 Transformer
**arXiv:2603.26380** | Authors: Yusheng Zhao et al. | cs.CL

**摘要**：标准全注意力的计算随序列长度平方增长，是长上下文语言建模的主要瓶颈。滑动窗口注意力以牺牲感受野为代价换取效率。现有混合模型尝试兼顾两者，但采用静态、启发式设计的交替模式，无法在多样化场景中高效分配计算。本文提出 Switch Attention (SwiAttn)，在每个 token、每层动态路由到全注意力分支（全局信息聚合）或滑动窗口分支（高效局部模式匹配）。设计自适应正则化目标鼓励模型趋于高效。持续预训练将全注意力架构迁移到混合架构。在 23 个基准数据集（4K 和 32K 上下文）上验证了有效性。

**PDF**：[https://arxiv.org/pdf/2603.26380](https://arxiv.org/pdf/2603.26380)

### 14. 对话音频抽象压缩用于 LLM-based ASR
**arXiv:2603.26246** | Authors: Shashi Kumar et al. | cs.CL / cs.AI / cs.LG

**摘要**：标准 LLM 语音识别系统孤立处理语音段，无法利用对话上下文。本文研究多模态上下文是否改善 LLM-based ASR 以及如何高效表示。监督多轮训练后发现，对话上下文主要帮助识别上下文实体。但原始上下文成本高——先前轮次的音频 token 序列随对话长度快速增长。本文提出 Abstract Compression，用固定数量的学习潜在 token 替换先前轮次的音频部分，同时显式保留对应转录。压缩模型以更小先验音频占用恢复了大部分原始上下文条件化的收益。

**PDF**：[https://arxiv.org/pdf/2603.26246](https://arxiv.org/pdf/2603.26246)

### 19. Tabular Foundation Models 条件密度估计基准
**arXiv:2603.26611** | Authors: Rafael Izbicki et al. | cs.LG / stat.ML

**摘要**：条件密度估计（CDE）——恢复给定表格协变量响应的完整条件分布——在异方差、多模态或不对称不确定性场景中至关重要。最新表格 Foundation Models（如 TabPFN、TabICL）天然产生预测分布，但作为通用 CDE 方法的有效性尚未系统评估。本文在39个真实数据集（训练规模从50到20000）、6种指标（密度精度、校准、计算时间）上基准测试三种表格基础模型与参数化、基于树的和神经 CDE 基线。在大多数数据集上，基础模型在 CDE 损失、对数似然和 CRPS 上达到最佳；小样本下校准有竞争力，但某些指标在大样本下落后于任务特定神经基线。SDSS DR18 光度红移案例研究中，使用50000训练样本的 TabPFN 超越所有在500000样本上训练的基线。

### 26. PAPO: 通过解耦优势归一化的过程感知策略优化
