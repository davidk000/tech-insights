## 1. WriteBack-RAG: 可训练的知识库训练框架
**arXiv:2603.25737** | cs.AI / cs.CL / cs.IR | [PDF](https://arxiv.org/pdf/2603.25737)

**核心发现**：提出 WriteBack-RAG，将知识库视为可训练组件。通过标记样本识别检索成功案例，将相关文档蒸馏为紧凑知识单元并建立索引。跨 4 种 RAG 方法、6 个基准、2 种 LLM 骨干，平均提升 **+2.14%**，且蒸馏知识可跨方法迁移。

**亮点**：离线预处理，兼容任意 RAG pipeline，无需修改已有系统。
**作者**：Yuxing Lu et al.

## 2. WildASR: 语音 Agent 场景的 ASR 鲁棒性诊断基准
**arXiv:2603.25727** | cs.AI / cs.MM | [PDF](https://arxiv.org/pdf/2603.25727)

**核心发现**：现有 ASR 系统在真实语音 Agent 场景中仍有严重且不均匀的性能退化，且模型鲁棒性**不跨语言/条件迁移**。关键风险：模型在部分/降级输入下会幻觉性生成流畅但完全错误的内容，带来下游 Agent 安全风险。提出 WildASR 多语言（四语言）诊断基准，沿三个轴分解 ASR 鲁棒性：环境退化、人口统计转移、语言多样性。

**亮点**：含三大实用诊断工具，指导生产部署决策。
**作者**：Geeyang Tay, Wentao Ma et al. (Amazon)

## 3. RC2: 跨模态循环一致性强化学习
**arXiv:2603.25720** | cs.AI / cs.CV | [PDF](https://arxiv.org/pdf/2603.25720)

**核心发现**：多模态模型的跨模态不一致（视觉/文本表征给出矛盾预测）是核心问题。提出 RC2，通过强制跨模态循环一致性（向后推理→切换模态→重建答案）获得密集、无标签的奖励，在无需额外训练下提升推理准确率 **+7.6 分**。

**亮点**：循环约束自动对齐内部表征，高级推理从结构性一致性中涌现，而非单纯Scaling。
**作者**：Zirui Zhang et al.

## 4. Agent Factories: 通用编程 Agent 的硬件优化极限
**arXiv:2603.25719** | cs.AI / cs.AR / cs.LG | [PDF](https://arxiv.org/pdf/2603.25719)

**核心发现**：评估通用编程 Agent（Claude Code Opus 4.5/4.6，无需硬件专门训练）在 HLS 硬件优化上的表现。两阶段流水线：Stage 1 分解子内核独立优化 + ILP 全局配置；Stage 2 启动 N 个专家 Agent 探索跨函数优化。1→10 Agent 扩展：**平均 8.27x 加速**，streamcluster 超 20x，kmeans 约 10x。

**亮点**：Scaling Agent 数量是 HLS 优化的有效手段，且最优设计往往不来自排名最高的 ILP 候选。
**作者**：Abhishek Bhandwaldar et al.

## 5. NLAHs: 自然语言 Agent Harness
**arXiv:2603.25723** | cs.CL / cs.AI | [PDF](https://arxiv.org/pdf/2603.25723)

**核心发现**：Agent 性能越来越依赖 "harness 工程"，但 harness 设计通常埋在控制器代码中难以迁移/比较。提出 NLAHs（自然语言 Agent Harness）+ IHR（智能 Harness 运行时），用自然语言表达 harness 行为，通过显式契约、持久化产物、轻量适配器执行。跨编码和计算机使用基准验证。

**亮点**：harness 设计从代码中解耦，成为可移植、可研究的科学对象。
**作者**：Linyue Pan et al.

## 6. S2D2: 扩散 LLM 的免训练自推测解码
**arXiv:2603.25702** | cs.CL | [PDF](https://arxiv.org/pdf/2603.25702) | [代码](https://github.com/phymhan/S2D2)

**核心发现**：扩散语言模型在少步加速时，标准置信度阈值解码脆弱。S2D2 利用 block-diffusion 模型退化为自回归（block size=1）的特性，让同一预训练模型同时充当 drafter 和 verifier，插入推测验证步骤。SDAR 上实现 **4.7x 加速**，LLaDA2.1-Mini 上比静态基线快 4.4x 且精度略高。

**亮点**：零训练成本，混合扩散（并行）+ AR（局部序列级 critic）解码轨迹。
**作者**：Ligong Han et al.

## 7. LLM 自改进：系统级技术综述
**arXiv:2603.25681** | cs.CL | [PDF](https://arxiv.org/pdf/2603.25681)

**核心发现**：提出自改进 LLM 的统一闭环框架，包含四紧密耦合过程：数据获取→数据选择→模型优化→推理细化，配合自主评估层。模型在各阶段均发挥核心作用：当数据生成器、选择信号、参数更新器、输出精炼器。

**亮点**：面向完全自改进 LLM 的系统性技术路线图，梳理各组件代表性方法及局限性。
**作者**：Haoyan Yang et al.

## 8. 深度研究 Agent 的结构化评估：范畴论视角
**arXiv:2603.25342** | cs.LG | [PDF](https://arxiv.org/pdf/2603.25342)

**核心发现**：现有 DRA 评估依赖 ad hoc 经验基准。提出范畴论形式化框架，将深度研究工作流建模为保结构映射（函子）的组合。设计 296 题四轴基准：顺序连通链遍历、V-结构拉回交集验证、拓扑排序强制、本体论证伪（Yoneda Probe）。11 个主流模型评估：**SOTA 仅 19.9% 平均准确率**——多跳结构综合仍是 Agent 的严峻挑战。

**亮点**：高级 Agent 在拓扑重排序和本体验证上匹配纯推理模型，但**几乎全部在多跳结构综合上崩溃**。
**作者**：Zhiquan Tan et al.

## 9. On-Policy 蒸馏失败模式与修复
**arXiv:2603.25562** | cs.LG / cs.AI / cs.CL | [PDF](https://arxiv.org/pdf/2603.25562)

**核心发现**：LLM 后训练中常用的 on-policy 蒸馏（OPD）在长视野场景下存在三个失败模式：token 信号不平衡、教师指导不可靠、tokenizer 不匹配。提出教师 top-K 局部支持匹配（截断 reverse-KL + top-p rollout sampling + 特殊 token mask）。单任务数学推理和多任务 agent+数学训练均获更稳定优化和更好下游性能。

**亮点**：从估量器和实现两侧重新审视 OPD，提供可复现的修复方案。
**作者**：Yuqian Fu et al.

## 10. LLM 作为 Token 压缩器/解压器
**arXiv:2603.25340** | cs.CL | [PDF](https://arxiv.org/pdf/2603.25340)

**核心发现**：提出自表达自编码学习框架，将预训练 LLM 微调为将长文本压缩为离散可变长潜在码（Z-tokens），并从 Z-tokens 精确重建原文。内容自适应：语义密集段获得更多 Z-tokens，可预测区域被激进压缩（LoRA 适配头）。Wikipedia 等数据集上实现 **最高 18 倍 token 压缩**，支持提示压缩和 Z-token 空间内自回归生成。

**亮点**：长上下文推理的 token 高效路径，18x 压缩下仍保留下游性能。
**作者**：Wenbing Li et al.

## 11. 跨模型不一致性：免标签正确性信号
**arXiv:2603.25450** | cs.AI | [PDF](https://arxiv.org/pdf/2603.25450)

**核心发现**：现有方法依赖模型自身不确定性（token 熵/置信度），在"自信错误"这一最危险失败模式下完全失效。提出 Cross-Model Perplexity (CMP) 和 Cross-Model Entropy (CME)：用第二个验证模型对生成模型答案做单次前向传播，测量惊讶程度。MMLU 上 AUROC 从 0.59 提升至 **0.75**。

**亮点**：免训练、免标签，直接插入生产路由/监控管线；覆盖推理、检索、数学基准。
**作者**：Matt Gorbett et al.

## 12. 剪枝语言模型的 SAE 分析
**arXiv:2603.25325** | cs.LG / cs.AI | [PDF](https://arxiv.org/pdf/2603.25325)

**核心发现**：首次系统研究无结构剪枝如何重塑语言模型的特征几何。跨 Gemma 3 1B、Gemma 2 2B、Llama 3.2 1B 三大模型族，两种剪枝方法，六种稀疏度。**反直觉发现**：稀有 SAE 特征（低触发率）比高频特征更耐剪枝，剪枝相当于隐式特征选择——优先破坏高频通用特征，保留专门的稀有特征。Wanda 剪枝比 magnitude 保留 3.7 倍特征结构。

**亮点**：几何特征存活率≠因果重要性，对压缩下可解释性有重要启示。
**作者**：Hector Borobia et al.

## 13. PICon: Agent Persona 一致性评估框架
**arXiv:2603.25620** | cs.CL | [PDF](https://arxiv.org/pdf/2603.25620) | [Demo](https://kaist-edlab.github.io/picon/)

**核心发现**：LLM persona agent 快速普及，但缺乏系统一致性验证方法。提出 PICon，通过逻辑链多轮追问探测 Agent，评估三维一致性：内部（无自相矛盾）、外部（与现实一致）、重测（重复下稳定）。63 名真实人类对比：即使先前报告"高度一致"的系统，在所有三维上**均未达到人类基线**，揭示矛盾和回避行为。

**亮点**：系统化追问暴露人工设定人格 Agent 的根本性局限。
**作者**：Sujeong Im et al. (KAIST)

## 14. LLM 在学术论文中的影响：词汇使用模式分析
**arXiv:2603.25638** | cs.CL / cs.AI / cs.CY / cs.DL / cs.LG | [PDF](https://arxiv.org/pdf/2603.25638)

**核心发现**：分析 arXiv 论文词汇变化：标题中 "beyond"/"via" 增加，"the"/"of" 减少；当前分类器难以精确判定多类别 LLM 来源；不同 LLM 导致学术论文词汇使用模式持续演化。采用线性可解释方法定量评估，现实 LLM 使用是异质且动态的。

**亮点**：可视化工具：https://llm-impact.github.io/word-usage-arxiv-abstract/
**作者**：Mingmeng Geng et al.
