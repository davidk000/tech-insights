# Day1：股票是什么？

**结论：股票是企业所有权的凭证，买股票就是当股东。**

## 核心概念
| 概念 | 解释 |
|:---|:---|
| 股票 | 企业公开发行募资时发的所有权凭证 |
| A股 | 在沪深交易所上市、用人民币交易的股票 |
| 上证指数 | 上海交易所全部上市公司，代码000001 |
| 深证成指 | 深圳交易所精选500家，代码399001 |
| 创业板 | 深圳的"创新企业板"，代码399006 |
| 科创板 | 上海的"硬科技板"，代码000688 |

## 为什么买股票能赚钱？
1. **股息分红**：公司盈利后分现金给你
2. **差价收益**：低买高卖赚取价差
3. **股本扩张**：公司成长推动股价长期上涨

## A股交易规则（必须记住）
| 项目 | 规则 |
|:---|:---|
| 交易时间 | 9:15-9:25集合竞价，9:30-11:30，13:00-15:00 |
| 涨跌幅 | 普通股±10%，ST股±5%，新股上市首日无限制 |
| 最少买100股 | 1手=100股，买入必须是100的整数倍 |
| T+1 | 当天买的股票，当天不能卖 |

## 三大指数
- **上证指数**：上海交易所全部股票，A股风向标
- **深证成指**：深圳交易所精选500家
- **创业板**：深圳创新企业板，波动最大
