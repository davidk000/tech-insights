## Day1：股票是什么？

**结论：股票是企业所有权的凭证，买股票就是当股东。**

### 核心概念
| 概念 | 解释 |
|:---|:---|
| 股票 | 企业公开发行募资时发的所有权凭证 |
| A股 | 在沪深交易所上市、用人民币交易的股票 |
| 上证指数 | 上海交易所全部上市公司，代码000001 |
| 深证成指 | 深圳交易所精选500家，代码399001 |
| 创业板 | 深圳的"创新企业板"，代码399006 |
| 科创板 | 上海的"硬科技板"，代码000688 |

### 为什么买股票能赚钱？
1. **股息分红**：公司盈利后分现金给你
2. **差价收益**：低买高卖赚取价差
3. **股本扩张**：公司成长推动股价长期上涨

### A股交易规则（必须记住）
| 项目 | 规则 |
|:---|:---|
| 交易时间 | 9:15-9:25集合竞价，9:30-11:30，13:00-15:00 |
| 涨跌幅 | 普通股±10%，ST股±5%，新股上市首日无限制 |
| 最少买100股 | 1手=100股，买入必须是100的整数倍 |
| T+1 | 当天买的股票，当天不能卖 |

## Day2：看懂股票行情

**结论：行情软件上的数字不是噪音，每一个都有含义。**

### 行情界面解读
```
贵州茅台  1416.02  ▲+1.06%  成交量:3.2万手  成交额:45.2亿
```

| 字段 | 含义 | 怎么看 |
|:---|:---|:---|
| 当前价 | 最新成交价 | 实时跳动 |
| 涨跌幅 | 相对昨收的变化% | ▲红=涨，▼绿=跌（A股反美股） |
| 成交量 | 成交股数（手=100股） | 放量=活跃，缩量=冷淡 |
| 成交额 | 成交量×价格 | 真实资金进出 |
| 换手率 | 成交股数/总流通股 | >5%活跃，<1%冷清 |

### 关键指标速查
| 指标 | 计算 | 含义 |
|:---|:---|:---|
| PE（市盈率） | 股价/每股收益 | 越低越便宜，<15合理，>30偏贵 |
| PB（市净率） | 股价/每股净资产 | <1破净，>3溢价 |
| 股本 | 总发行股数 | 越大越稳，越小越易被炒作 |

### A股特色板块
- **沪深300**：沪深两市最大的300只蓝筹股
- **中证500**：中等市值代表
- **北向资金**：外资通过沪/深港通买入A股的金额
- **融资融券**：杠杆资金，情绪放大器

## Day3：财务报表入门

**结论：选股就是选公司，公司赚钱才是真的。**

### 三张表速读

**1. 资产负债表（家底）**
```
资产 = 负债 + 股东权益
```
看负债率（<50%稳健，>70%危险）

**2. 利润表（赚钱能力）**
```
营业收入 → 营业成本 → 毛利润 → 净利润
```
核心看净利润增长，连续2年增长=成长股

**3. 现金流量表（真金白银）**
```
经营现金流 > 0  ✅  主业真正赚钱
投资现金流 < 0  ✅  在扩张
融资现金流 < 0  ✅  在还债或分红
```

### 最核心的4个数字
| 指标 | 公式 | 及格线 |
|:---|:---|:---|
| ROE（净资产收益率）| 净利润/净资产 | >15%优秀 |
| 毛利率 | (营收-成本)/营收 | >30%有护城河 |
| 净利润增速 | (今年-去年)/去年 | >10%成长 |
| 负债率 | 总负债/总资产 | <60%稳健 |

### 实战：茅台为什么是A股之王？
- ROE：持续>30%（钱生钱能力极强）
- 毛利率：90%+（酱香护城河）
- 现金流：年年为正
- 负债率：<30%

### 第三天作业
- [ ] 找一家你感兴趣的公司，看它的PE和ROE
- [ ] 理解三个财务报表各自在说什么
- [ ] 记住：现金流比利润更重要

## Day4：估值与安全边际

**结论：好公司还要有好价格，再好的股票买贵了也是风险。**

### 估值方法
| 方法 | 适用 | 公式 |
|:---|:---|:---|
| PE法 | 成熟企业 | PE < 15低估，> 30高估 |
| PEG法 | 成长企业 | PEG = PE/增长率，<1便宜 |
| PB法 | 周期/金融 | PB < 1破净 |
| DCF法 | 所有企业 | 未来现金流折现 |

### 安全边际
> "用4毛钱买1块钱的东西" — 格雷厄姆

安全边际 = 内在价值 - 当前价格
- 折扣>30%：极佳买点
- 折扣10-30%：合理范围
- 溢价：等待

### 价值投资的两种路径
1. **捡烟蒂**：找低估股票，等估值修复（格雷厄姆）
2. **成长股**：找优质公司，长期持有（巴菲特）

### A股常见陷阱
- **低PE陷阱**：强周期股（钢铁/煤炭）低PE反而是高点
- **高增长陷阱**：并购带来的假增长，一次性收益
- **政策题材炒作**：没有业绩支撑的概念，最终归零

- [ ] 理解为什么"低估"不等于"马上涨"
- [ ] 记住：安全边际是投资的护身符

## Day5：技术分析入门

**结论：K线是市场情绪的轨迹，辅助判断但不能预测未来。**

### K线基础
```
阳线（红）= 收盘>开盘 = 买盘强
阴线（绿）= 收盘<开盘 = 卖盘强
上下影线 = 最高/最低点相对开盘/收盘的位置
```

### 三个最重要的指标

**1. 均线（MA）**
- MA5：5日均线，短期趋势
- MA20：月线，中期趋势
- MA60：季线，长期趋势
- 金叉：MA5上穿MA20 → 买入信号
- 死叉：MA5下穿MA20 → 卖出信号

**2. MACD**
- DIF线：快线
- DEA线：慢线
- 红柱/绿柱：动能指示
- DIF上穿DEA → 买入

**3. 量价关系**
- 放量上涨 = 真实涨势（可能延续）
- 缩量上涨 = 虚涨（随时回调）
- 放量下跌 = 恐慌抛售（可能见底）
- 地量横盘 = 变盘前兆

### A股技术分析注意
- A股T+1：当天买的不能卖，技术信号会滞后
- 主力资金可以做出"骗线"
- 技术分析是概率游戏，不是预言

- [ ] 理解MACD的金叉死叉
- [ ] 记住：量先于价，量是关键

## Day6：交易策略与心态

**结论：赚钱靠策略，亏钱靠心态。**

### 常见策略
| 策略 | 做法 | 适合人群 |
|:---|:---|:---|
| 定投 | 每月固定金额买指数基金 | 工作忙、没时间 |
| 趋势跟踪 | 金叉买，死叉卖 | 有时间看盘 |
| 价值投资 | 低估值时买入，长期持有 | 有耐心 |
| 网格交易 | 设定价格区间，低买高卖 | 震荡市 |

### 定投沪深300策略（最适合新手）
```
每月第一个交易日，买入沪深300ETF（510300）
金额：每月闲钱的20%
目标：持有3-5年，等待牛市
```

### 仓位管理
| 行情 | 仓位建议 |
|:---|:---|
| 大牛市 | 80-100% |
| 正常 | 50-70% |
| 大熊市/极度悲观 | 20-30% |
| 完全不确定 | 空仓 |

### 必须养成的习惯
- **每日复盘**：收盘后花5分钟看今日涨跌原因
- **止损线**：单只股票亏损>8%必须重新评估
- **不追涨杀跌**：90%的散户亏钱都是因为这个
- **分散持仓**：不超过5只，单只不超总仓位30%

### 第六天作业
- [ ] 设定自己的止损线（建议8%）
- [ ] 决定自己的投资风格（定投/趋势/价值）
- [ ] 记住：亏钱时不平仓，是大多数人的致命错误

## Day7：开始行动

**结论：理解≠做到，从模拟盘或最小仓位开始。**

### 现在可以做什么
1. **开个证券账户**：中信/华泰/国泰君安都是头部券商
2. **先看不动**：开户后第一个月，只看不买
3. **最小仓位**：用1万块钱以内体验真实涨跌
4. **记录每日感受**：涨了为什么？跌了为什么？
