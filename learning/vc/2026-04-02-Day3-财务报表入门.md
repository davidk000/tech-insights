# Day3：财务报表入门

**结论：选股就是选公司，公司赚钱才是真的。**

---

## 三张表速读

**1. 资产负债表（家底）**
```
资产 = 负债 + 股东权益
```
看负债率（<50%稳健，>70%危险）

**2. 利润表（赚钱能力）**
```
营业收入 → 营业成本 → 毛利润 → 净利润
```
核心看净利润增长，连续2年增长=成长股

**3. 现金流量表（真金白银）**
```
经营现金流 > 0  ✅  主业真正赚钱
投资现金流 < 0  ✅  在扩张
融资现金流 < 0  ✅  在还债或分红
```

---

## 最核心的4个数字

| 指标 | 公式 | 及格线 |
|:---|:---|:---|
| ROE（净资产收益率）| 净利润/净资产 | >15%优秀 |
| 毛利率 | (营收-成本)/营收 | >30%有护城河 |
| 净利润增速 | (今年-去年)/去年 | >10%成长 |
| 负债率 | 总负债/总资产 | <60%稳健 |

---

## 实战：茅台为什么是A股之王？

- ROE：持续>30%（钱生钱能力极强）
- 毛利率：90%+（酱香护城河）
- 现金流：年年为正
- 负债率：<30%
