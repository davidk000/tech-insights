# Day5：技术分析入门

**结论：K线是市场情绪的轨迹，辅助判断但不能预测未来。**

---

## K线基础

```
阳线（红）= 收盘>开盘 = 买盘强
阴线（绿）= 收盘<开盘 = 卖盘强
上下影线 = 最高/最低点相对开盘/收盘的位置
```

---

## 三个最重要的指标

### 1. 均线（MA）
- **MA5**：5日均线，短期趋势
- **MA20**：月线，中期趋势
- **MA60**：季线，长期趋势
- **金叉**：MA5上穿MA20 → 买入信号
- **死叉**：MA5下穿MA20 → 卖出信号

### 2. MACD
- **DIF线**：快线
- **DEA线**：慢线
- **红柱/绿柱**：动能指示
- **DIF上穿DEA → 买入**

### 3. 量价关系
- **放量上涨** = 真实涨势（可能延续）
- **缩量上涨** = 虚涨（随时回调）
- **放量下跌** = 恐慌抛售（可能见底）
- **地量横盘** = 变盘前兆
