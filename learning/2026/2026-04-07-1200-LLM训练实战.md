## 📖 完整讲解

## 原理

学习率是训练最重要的超参之一。设置不当，轻则收敛慢，重则 loss 爆炸。

| 策略 | 公式 | 适用场景 |
|:---|:---|:---|
| Linear Warmup | 0 → peak → const | 大多数场景 |
| Cosine Annealing | lr = peak × cos(πt/T) | 长期训练 |
| Warmup + Cosine | 先升后降 | LLM 训练标配 |

**Warmup 至关重要**：训练初期权重随机，如果学习率太大，梯度方向震荡剧烈，模型容易崩溃。通常 Warmup 1000~2000 steps。

```python
scheduler = CosineAnnealingWarmRestarts(
    optimizer, T_0=10, T_mult=2, eta_min=1e-6
)
```


LLaMA 的学习率调度：先 Linear Warmup（2000 steps），再 Cosine Annealing 到 10% 的峰值。
