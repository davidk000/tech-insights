## 📖 完整讲解

## 原理

Triton kernel 可以像普通 PyTorch Module 一样使用。

### 三种集成方式

方式1：torch.autocast 混合精度
```python
with torch.autocast(device_type='cuda', dtype=torch.float16):
    output = triton_kernel(input)
```

方式2：torch.compile 加速
```python
compiled_kernel = torch.compile(triton_kernel)
output = compiled_kernel(input)
```

方式3：自定义 CUDA Extension
```python
from torch.utils.cpp_extension import load_inline
src = "..."
module = load_inline(name='triton_softmax_ext', cpp_sources=src, extra_cuda_cflags=['-O3'])
```

### 实际使用场景

Triton 主要用于：
1. 自定义 Attention 变体（如 Flash Attention 的某些配置）
2. 新算子的快速原型验证
3. 替代无法用 PyTorch 实现的 CUDA kernel

大多数情况下，直接用 `torch.nn.functional` 或第三方库（Flash Attention、vLLM）就够了，不需要自己写 Triton。
