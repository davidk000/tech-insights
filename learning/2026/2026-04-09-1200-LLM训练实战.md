## 📖 完整讲解

## 原理

SFT（Supervised Fine-Tuning）= 在预训练模型基础上，用有标签数据继续训练。

预训练学常识（海量无标签文本），SFT 学任务能力（有标签数据）。

### 两种主流方式对比

| 方式 | 显存需求 | 速度 | 效果 |
|:---|:---|:---|:---|
| 全量微调 | 巨大（65B ~780GB）| 慢 | 最好 |
| **LoRA** | 极低（65B ~48GB）| 快 | 略低于全量 |
| **QLoRA** | 更低（65B ~48GB，4-bit）| 中 | 接近全量 |

LoRA 核心：冻结预训练权重，只训练两个低秩矩阵 A 和 B。

```python
class LoRALinear(nn.Module):
    def __init__(self, linear, rank=4, alpha=4):
        super().__init__()
        self.linear = linear
        self.lora_A = nn.Parameter(linear.weight.new_zeros(rank, linear.in_features))
        self.lora_B = nn.Parameter(linear.weight.new_zeros(linear.out_features, rank))

    def forward(self, x):
        return self.linear(x) + (x @ self.lora_A.T @ self.lora_B.T) * (self.alpha / self.rank)
```

QLoRA = NF4 量化基座（4-bit NormalFloat）+ LoRA = 65B 模型单卡可微调。
