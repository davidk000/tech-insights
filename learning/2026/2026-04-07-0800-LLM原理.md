## 📖 完整讲解

## 原理

GPT 和 BERT 是 Transformer 架构的两种代表性分支，分别走向了不同的技术路线。

### 核心区别

| | GPT | BERT |
|:---|:---|:---|
| 架构 | 只用 Decoder | 只用 Encoder |
| 训练方式 | 自回归（Predict Next Token）| 自编码（Masked LM）|
| 注意力 | 因果注意力（只看前文）| 双向注意力（看全文）|
| 擅长 | 文本生成、对话、代码 | 理解任务：分类、抽取、问答 |
| 代表模型 | GPT-2/3/4, LLaMA, ChatGPT | BERT, RoBERTa, DeBERTa |

### GPT 的自回归训练

GPT 训练的目标是「预测下一个词」：
```
输入: "今天天气真"
输出: "好"
```
Loss 只在「下一个词」的位置计算。模型只能看到前文，不能偷看答案。

### BERT 的 Masked LM 训练

BERT 训练的目标是「预测被遮住的词」：
```
输入: "今天天气真[MASK]"
输出: "好"
```
模型可以看到全文（双向注意力），所以能更好地理解上下文语义。

## 常见误区

1. "BERT 已经过时了" → 错，BERT 在 NER、问答、语义匹配上仍是主流 backbone
2. "GPT 不能做理解任务" → 能，但需要用 Prompt Engineering 引导
3. "BERT 比 GPT 更好" → 任务不同，架构选择也不同
