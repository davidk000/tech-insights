**Self-Attention 详解：Q/K/V 向量/点积注意力/Scaled Dot-Product/Softmax**

---

## 📖 完整讲解

## 原理

Self-Attention（自注意力）是 Transformer 的心脏。核心问题：序列中每个 token 如何知道其他 token 的重要性？

传统 RNN 按顺序处理，长距离依赖靠 hidden state 传递，信息衰减严重。Self-Attention 的思路：并行计算任意两个位置的关联强度。

每个输入 token 先投影成三个向量：
- **Q（Query）**：我在找什么
- **K（Key）**：我包含什么信息
- **V（Value）**：如果匹配成功，我贡献什么

注意力公式：`Attention(Q,K,V) = softmax(Q·K^T / √d) · V`，其中 `√d` 是缩放因子，防止点积过大导致 Softmax 梯度消失。

## 代码示例

```python
import torch
import torch.nn.functional as F
import math

def self_attention(Q, K, V, mask=None):
    d_k = Q.size(-1)
    scores = torch.matmul(Q, K.transpose(-2,-1)) / math.sqrt(d_k)
    if mask is not None:
        scores = scores.masked_fill(mask == 0, -1e9)
    attn = F.softmax(scores, dim=-1)
    return torch.matmul(attn, V), attn

Q = torch.randn(2, 8, 512)  # batch=2, heads=8, seq=512
K = torch.randn(2, 8, 512)
V = torch.randn(2, 8, 512)
out, w = self_attention(Q, K, V)
print(out.shape)  # torch.Size([2, 8, 512])
```

## 类比

想象你在读论文：读到某段时，你会回头找相关段落（查 Key），判断相关程度（Q·K），然后提取关键信息（V）。Self-Attention 就是让模型同时做这件事。

## 常见误区

| 误区 | 正确理解 |
|:---|:---|
| Q/K/V 是独立可学习的参数 | 它们是输入的线性投影，投影矩阵 W 才是参数 |
| 注意力越强越好 | 某些位置（如标点）不应该有高注意力权重 |
| Self-Attention 不需要位置编码 | 它不包含位置信息，必须额外加位置编码 |
