## 学习目标

通过 21 天高强度学习，建立以下领域的基础到进阶知识体系：

| 课程 | 时长 | 目标 |
|:---|:---:|:---|
| 1. LLM 原理 | 14天 | 掌握 Transformer → Attention → MoE → RLHF |
| 2. LLM 训练实战 | 14天 | 分布式训练/DeepSpeed/RLHF/数据工程 |
| 3. LLM 推理优化 | 14天 | Flash Attention / PagedAttention / vLLM / TensorRT |
| 4. Triton 编程 | 7天 | CUDA 替代/自定义Kernel/AI编译器基础 |
| 5. 编译原理 | 7天 | LLVM架构/中间表示/优化遍 |
| 6. 超节点架构 | 7天 | NPU/GPU异构/Chiplet/高速互联/内存墙 |
| 7. 云原生架构 | 7天 | K8s Operator/ServiceMesh/eBPF/边缘计算 |
| 8. 托业考试 | 7天 | 词汇积累+阅读技巧突破 |

## 课程详细大纲

### 课程1: LLM原理（14节，08:00档）

| 节次 | 内容 |
|:---:|:---|
| L1-1 | Transformer 诞生背景：RNN缺陷/注意力机制起源/《Attention Is All You Need》|
| L1-2 | Self-Attention 详解：Q/K/V向量/点积注意力/Scaled Dot-Product |
| L1-3 | Multi-Head Attention：多头并行/残差连接/LayerNorm |
| L1-4 | 位置编码：Sinusoidal/RoPE/ALiBi |
| L1-5 | GPT系列：单向Attention/下一个Token预测 |
| L1-6 | BERT系列：双向掩码/MLM |
| L1-7 | KV Cache：显存占用/加速Decode原理 |
| L1-8 | MoE：稀疏激活/GShard/Mixtral/负载均衡 |
| L1-9 | 长上下文：FlashAttention/RoPE插值/位置编码外推 |
| L1-10 | RLHF：InstructGPT三阶段/奖励模型/PPO |
| L1-11 | DPO：直接偏好优化/无需PPO |
| L1-12 | CoT/ReAct：思维链/推理行动协同 |
| L1-13 | 长思考模型(o1/o3)：隐式推理/PRM/测试时计算 |
| L1-14 | LLM架构选型：开源模型对比/LLaMA/Mistral/Qwen |
