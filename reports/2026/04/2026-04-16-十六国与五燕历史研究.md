## 材料二：五燕传承关系与世系表

> **五燕**：前燕、后燕、西燕、南燕、北燕。皆与鲜卑慕容氏有直接或间接渊源。西燕不在十六国之列；北燕实为汉人（冯跋）政权，非慕容氏。

### 五燕总览

| 国号 | 存续 | 族属 | 开国君主 | 亡于 | 备注 |
|:---|:---|:---|:---|:---|:---|
| 前燕 | 337—370 | 鲜卑慕容部 | 慕容皝 | 前秦 | 慕容氏诸燕之首 |
| 西燕 | 384—394 | 鲜卑慕容部 | 慕容泓 | 后燕 | 不在十六国之列 |
| 后燕 | 384—409 | 鲜卑慕容部 | 慕容垂 | 北燕 | 淝水之战后慕容垂所建 |
| 南燕 | 398—410 | 鲜卑慕容部 | 慕容德 | 东晋 | 后燕分裂而来 |
| 北燕 | 409—436 | 汉（冯跋/高云） | 冯跋 | 北魏 | 实为汉人政权，慕容氏已绝 |

### 前燕世系（337—370）

```
慕容廆（武宣帝，追尊）
  └─ 慕容皝（文明皇帝，334-348年实际统治）
        └─ 慕容儁（景昭皇帝，349-360年）
              ├─ 慕容𬀩（幽皇帝，360-370年）→ 370年亡于前秦
              └─ 慕容垂（后燕开国皇帝）
```

### 西燕世系（384—394，不在十六国）

> 384年，前燕遗族慕容泓起兵反秦，386年慕容冲攻入长安，此后段随、慕容𫖮、慕容瑶、慕容忠相继被杀，386年慕容永最终稳定局势。394年后燕灭西燕。

```
慕容暐（前燕亡国之君，384年死）
  ├─ 慕容泓（384年）→ 济北王，被杀
  ├─ 慕容冲（384-386年）→ 杀泓自立，被杀
  ├─ 段随（386年）→ 杀冲自立，被杀
  ├─ 慕容𫖮（386年）→ 被杀
  ├─ 慕容瑶（386年）→ 被杀
  ├─ 慕容忠（386年）→ 被杀
  └─ 慕容永（386-394年）→ 河东王→帝，394年被后燕所杀
```

### 后燕世系（384—409）

> 后燕原为慕容垂所建，407年高云（慕容云）杀慕容熙夺位，仍沿用燕国号，史学界以407年为后燕亡/北燕始。

```
慕容垂（成武皇帝，384-396年）
  └─ 慕容宝（惠愍皇帝，396-398年）
        ├─ 慕容详（398年）→ 自立，被杀
        ├─ 慕容麟（398年）→ 自立，逃降北魏
        └─ 兰汗（398年）→ 杀宝自立，被慕容盛杀
              └─ 慕容盛（昭武皇帝，398-401年）
                    └─ 慕容熙（昭文皇帝，401-407年）
                          └─ 慕容云/高云（407-409年）→ 被冯跋杀
```

### 南燕世系（398—410）

> 397年后燕被北魏截断，慕容德率部南迁，398年于滑台称王，400年称帝，都广固。410年东晋刘裕攻灭南燕。

```
慕容廆
  └─ 慕容皝
        └─ 慕容儁
              ├─ 慕容垂（后燕）
              └─ 慕容德（献武皇帝，398-405年）
                    └─ 慕容超（太上皇帝，405-410年）→ 410年亡于东晋刘裕
```

### 北燕世系（409—436）

> 北燕实为汉人政权。407年冯跋辅佐高云（慕容云，慕容宝养子，高句丽系）夺位，409年高云被杀，冯跋继立，436年北魏灭北燕。

```
高云（慕容云）407-409年（被冯跋杀）
  └─ 冯跋（昭文皇帝，409-430年）
        └─ 冯弘（430-436年）→ 436年亡于北魏
```

**君主列表：**

|:---|:---|:---|:---|

| 冯跋 | 汉 | 409-430 | 太平 |
| 冯弘 | 汉 | 430-436 | 太兴 |
