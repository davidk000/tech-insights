# 自我进化机制 (Self Evolution)

## 三层记忆体系

```
┌─────────────────────────────────────┐
│  L1 工作记忆 (Working Memory)        │
│  - 当前会话上下文                     │
│  - 临时存储，会话结束清空              │
└──────────────┬──────────────────────┘
               ↓ 重要事件
┌─────────────────────────────────────┐
│  L2 短期记忆 (Short-term Memory)     │
│  - 每日记忆文件                       │
│  - memory/YYYY-MM-DD.md              │
│  - 7天轮换                           │
└──────────────┬──────────────────────┘
               ↓ 核心经验
┌─────────────────────────────────────┐
│  L3 长期记忆 (Long-term Memory)      │
│  - 永久记忆文件                       │
│  - MEMORY.md                          │
│  - 核心知识、偏好、决策               │
└─────────────────────────────────────┘
```

## 每日进化任务（22:00）

### 执行步骤

1. **回顾会话历史**
   - 读取当天的会话记录
   - 提取关键事件和决策
   - 识别学到的新东西

2. **压缩整理记忆**
   - 将重要信息保存到短期记忆文件
   - 格式：`memory/YYYY-MM-DD.md`

3. **分析总结**
   - 今天学会的新技能
   - 犯的错误和解决方法
   - 可以改进的地方

4. **生成进化报告**
   - 提议可以固化的3个技能
   - 明日学习计划
   - 成长指标统计

5. **发送报告**
   - 通过消息发送给用户

## 固化技能标准

满足以下条件考虑固化为skill：
- [ ] 重复使用 **3次以上**
- [ ] 解决**通用问题**
- [ ] 可以被**标准化**
- [ ] 有明确的输入输出

| 指标 | 目标 | 当前 |
|:---|:---:|:---:|
| 学习技能 | 1个/天 | - |
| 固化技能 | 1个/周 | - |
| 错误减少 | 50%/月 | - |
| 效率提升 | 20%/月 | - |
| 知识整合率 | 30% | - |

## 每周进化任务（周日）

1. 回顾本周所有短期记忆
2. 提取核心经验到 MEMORY.md
3. 清理7天前的短期记忆文件
4. 生成本周成长报告
5. 制定下周学习目标

## 每月进化任务

1. 评估成长进度
2. 调整成长策略
3. 生成月度报告
4. 更新技能知识库
5. 优化工作流程
