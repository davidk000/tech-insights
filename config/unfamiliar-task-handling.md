# 陌生任务处理流程 (Unfamiliar Task Handling)

## 学习来源优先级

| 优先级 | 来源 | 时间分配 | 说明 |
|:---|:---:|:---:|:---|
| **P1** | ClawHub技能库 | 50% | 现成的skill或工具 |
| **P2** | GitHub开源项目 | 30% | 开源代码和方案 |
| **P3** | YouTube/B站视频 | 15% | 视频教程和字幕 |
| **P4** | 其他来源 | 5% | 文档、博客、社区 |

## 学习流程

```
识别陌生任务
    ↓
1. 搜索ClawHub
    ↓
有 → 安装skill → 深度学习skill → 应用
    ↓
无 → 2. 搜索GitHub
    ↓
有 → 学习代码 → 创建自定义skill → 应用
    ↓
无 → 3. 搜索YouTube/B站
    ↓
有 → 学习教程 → 实践 → 固化skill
    ↓
无 → 4. 搜索引擎/文档
    ↓
学习 → 实践 → 固化skill
```

## 深度学习skill步骤

1. **安装skill**
   ```bash
   clawhub install <skill-name>
   ```

2. **阅读skill内容**
   - 查看目录结构
   - 阅读 SKILL.md
   - 阅读主要代码文件

3. **提取知识点**
   - 核心方法
   - 最佳实践
   - 设计模式
   - 使用技巧

4. **整合知识**
   - 关联已有知识
   - 发现共同模式
   - 创造新方法

5. **记录到知识库**
   - 保存到 `data/skill-knowledge-base.json`

## 学习原则

- **快速学习**：Token消耗 < 总Token的20%
- **够用即可**：找到关键知识即可，不求完美
- **固化skill**：将解决方案固化为skill，避免重复学习

## 技能组合

当掌握多个相关skill后，尝试组合创造新能力：

| 组合 | 新能力 | 示例 |
|:---|:---|:---|
| image + prompt + social | 自动配图发帖 | 生成图片+文案+发布 |
| search + analysis + report | 调研自动化 | 搜索+分析+生成报告 |
| calendar + reminder + message | 智能助理 | 日程+提醒+通知 |
