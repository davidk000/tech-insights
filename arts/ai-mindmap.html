人工智能基础
一、AI基础概念
定义：模拟、延伸、拓展人类智能的交叉学科
流派分类
符号主义 → 逻辑推理（如ES、KN）
连接主义 → 神经网络（如NN）
行为主义 → 环境交互（如RL）
发展历史：1956年达特茅斯会议正式提出AI，历经三起两落发展周期
二、三大核心要素
算力：NPU / GPU / TPU芯片，决定模型运行速度
数据：数据质量、数量直接决定模型效果上限
算法：ML、DL、RNN、CNN、Transformer数学建模结构
三、机器学习核心
学习分类
有监督学习：标签数据分类识别
无监督学习：无标签自动聚类
强化学习：环境交互（自动驾驶、游戏AI）
两大环节
训练：挖掘数据规律、拟合模型、调参优化
→
推理：用训练好模型执行预测任务
完整落地流程
数据准备
→
特征工程
→
模型训练
→
调优
→
压缩部署
四、神经网络原理
基础结构：输入层 → 隐藏层 → 输出层（全连接网络）
经典网络区别
CNN图像视觉处理
RNN序列时序数据
Transformer自然语言NLP
运行逻辑：前向传播计算 → 反向传播更新 → 激活函数非线性拟合
